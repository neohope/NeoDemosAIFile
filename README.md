What is this project about
=========
This is just a project of AI file demos. 


How to build
============
1. install python 3.10+


2. update pip
```shell
    python -m pip install --upgrade pip
```

3. install packages
```shell
    cd subdir_xxx
    python -m venv venv
    . venv/bin/activate
    pip install -r requirements.txt
    python xxx.py
```

4. run
```shell
    cd subdir_xxx
    python -m venv venv
    python xxx.py
```
